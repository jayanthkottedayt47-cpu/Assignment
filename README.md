# Component Detection Dashboard

A simple React dashboard that allows users to upload a diagram image and view a list of circuit components. The application demonstrates React fundamentals, component structure, and responsive UI design.

---

## Features

* Upload diagram images (PNG/JPG)
* Image preview with file name display
* Zoom controls (Zoom In, Zoom Out, Reset)
* Sidebar showing circuit components
* Highlight selected component
* Responsive dashboard layout

---

## Tech Stack

* React
* Vite
* JavaScript (ES6)
* CSS

## Installation

Clone the repository:

```
git clone https://github.com/jayanthkottedayt47-cpu/Assignment.git
```

Navigate to the project folder:

```
cd component-detection-dashboard
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
